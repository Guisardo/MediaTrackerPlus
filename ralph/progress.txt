### Learnings for Future Iterations
- Tailwind v4 uses CSS cascade `@layer` internally — unlayered CSS from SCSS/CSS files ALWAYS beats layered Tailwind utilities. Move global base/reset styles into `@layer base` in tailwind.css to restore proper cascade ordering (replaces `important: true`)

- `@custom-variant dark (&:where(.dark, .dark *))` exactly replaces `darkMode: 'class'` from v3
- CSS `@source` directive replaces the `content` array from tailwind.config.js — path is relative to the CSS file location
- Tailwind v4 `@utility` blocks replace `plugin(({ addUtilities }) => { ... })` from v3 config
- `@theme { --font-sans: ... }` replaces `theme.extend.fontFamily` from v3 config
